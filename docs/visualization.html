visualize_static_orbits

 visualize_static_orbits (data:numpy.ndarray,
                          time_instants:Union[List[int],NoneType]=None,
                          orbit_indices:Union[List[int],NoneType]=None,
                          point_dict:Union[Dict[str,tuple],NoneType]=None,
                          show_legend:bool=True,
                          save_path:Union[str,NoneType]=None,
                          plot_reference_box:bool=True)

Visualizes orbits in 3D space and highlights specified time instants for each selected orbit.

Type Default Details
data ndarray The orbit data with shape (num_orbits, 6, num_time_points).
time_instants typing.Union[typing.List[int], NoneType] None Time points to highlight; defaults to None.
orbit_indices typing.Union[typing.List[int], NoneType] None Indices of orbits to visualize; defaults to all.
point_dict typing.Union[typing.Dict[str, tuple], NoneType] None Dictionary of extra points to plot.
show_legend bool True Flag to indicate whether to show a legend.
save_path typing.Union[str, NoneType] None Path to save the figure; defaults to None.
plot_reference_box bool True Flag to indicate whether to plot the reference box.
Returns None

point_dict typing.Union[typing.Dict[str, tuple], NoneType] None Named points as a dict with 3D coordinates.
filename str orbits.html Path and name of the file to save the HTML plot.
Returns None

show_labels typing.Union[bool, typing.List[bool]] True Option to print or not print labels.
percentage_font_size int 10 Font size for percentages.
label_distance float 1.1 Distance of labels from center.
pct_distance float 0.85 Distance of percentages from center.
explode_factor float 0.1 Factor to separate slices.
